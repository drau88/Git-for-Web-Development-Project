1. What is Semantic HTML? 
   
 Semantic HTML provides meaningful structure to the contents of a webpage. Examples of semantic HTML elements include <table>,<nav> and <section>.

 2. What is HTML used for?

 HTML is used for formatting the content that will be displayed on a webpage when it is read by a browser.

3. What is an attribute and where do we put it?

 An attribute is something that can be used within an HTML element to provide extra information or functionality to that element. An attribute is placed within the start tag, such as with the src element of the <img> element.

4. What is the h1 tag used for? How many times should I use it on a page?

 The h1 tag is used to declare the main or generally most important heading on a webpage. While it is possible to use h1 more than once, it is considered good practice to only use one h1 tag per page.

5. Name two tags that have required attributes

 Two tags that require attributes are <a>, which requires the href attribute, and <img>, which requires the src attribute.

7. What is an id?

 id is an attribute that provides an HTML element with a unique identifer which can then be specifically referenced for use in other languages or within other HTML elements on the same page.

8. What elements can I add an id to? 

 Id is a global attribute and can be used on any HTML element.

9. How many times can I use the same id on a page? 

 A unique id can appear only one time per webpage.

10. What is a class? 

 Class is an attribute similar to id, which provides HTML elements with an identifier. Unlike id, however, class can be used to identify more than one HTML element.

11. What elements can I add a class to? 

 Class is a global attribute that can be used with any HTML element.

12. How many times can I use the same class on a page? 

 Theoretically, as many elements as desired can belong to the same class on a webpage.

13. How do I get my link to open in a new tab?

 A link can be made to open in a new broswer tab by using target="_blank" within an <a> element.

14. What is the alt attribute in the image tag used for? 

 The alt attribute specifies alternative text that will be displayed on a webpage if the image does not render on the page.
